# atomskills

## Совместное редактирование документов

Система поддерживает реальное совместное редактирование документов через ONLYOFFICE. Пользователи могут работать над одним документом параллельно, видя курсоры друг друга и изменения в реальном времени.

### Как использовать

1. Запустите приложение с помощью `docker-compose up`.
2. Войдите в систему под учётной записью пользователя (например, Вадим или Андрей).
3. Перейдите на страницу профиля (Profile).
4. Система автоматически создаст или загрузит общий документ "Shared Document".
5. Откройте ту же страницу в другом браузере (или другом устройстве) под другим пользователем.
6. Оба пользователя увидят один и тот же документ и смогут редактировать его одновременно.

### Технические детали

- Для совместного редактирования используется ONLYOFFICE Document Server, запускаемый как отдельный контейнер.
- Документы хранятся в MinIO (S3-совместимое хранилище).
- Версии документов сохраняются в базе данных PostgreSQL.
- Редактор встраивается через iframe с конфигурацией, получаемой от бэкенда.
- Ключ документа (`key`) генерируется на основе ID документа и версии, что обеспечивает стабильную сессию совместного редактирования.

### Настройка

Убедитесь, что в файле `.env` заданы корректные переменные:
- `ONLYOFFICE_JWT_SECRET` — секрет для подписи JWT-токенов (должен совпадать с настройками ONLYOFFICE).
- `ONLYOFFICE_PUBLIC_URL` — публичный URL ONLYOFFICE Document Server (по умолчанию `http://localhost:8082`).

### Примечания

- Для работы совместного редактирования необходимо, чтобы оба пользователя имели доступ к одному и тому же документу (система предоставляет общий документ "Shared Document").
- Изменения сохраняются автоматически через callback-механизм ONLYOFFICE.
- Курсоры и имена других редакторов отображаются внутри ONLYOFFICE редактора.
